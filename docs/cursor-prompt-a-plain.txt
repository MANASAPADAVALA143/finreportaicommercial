MODULE 1: PVM Analysis  →  Route: /fpa/pvm
─────────────────────────────────────────

FRONTEND (src/pages/fpa/PVMAnalysis.tsx):
- Input form: Actual Revenue, Budget Revenue, Prior Year Revenue
- Table input: by Product (name, actual units, actual price, budget units, budget price)
- Table input: by Region (same structure)
- "Run PVM Analysis" button → POST /fpa/pvm-analysis
- Output section:
  → 3 KPI cards: Price Effect (£/$ value + %), Volume Effect, Mix Effect
  → Waterfall bar chart (recharts): Budget → Price → Volume → Mix → Actual
  → AI Commentary box (Claude 4.7 narrative, 3 paragraphs)
  → Download: Excel + PDF buttons

BACKEND (routers/fpa_pvm.py):
- Standard PVM formula:
  Price Effect  = Σ (Actual Price - Budget Price) × Actual Volume
  Volume Effect = Σ (Actual Volume - Budget Volume) × Budget Price  
  Mix Effect    = Total Variance - Price Effect - Volume Effect
- Call Claude API with structured PVM results for CFO-ready narrative
- Return: {price_effect, volume_effect, mix_effect, total_variance, by_product[], by_region[], commentary, waterfall_data[]}
